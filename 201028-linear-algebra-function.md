## 선형대수학
- PCA(Principal Component Analysis) : 주성분 분석
 - 차원축소와 투영 == 특이분해의 로우-랭크 근사문제
 - 근사 성능을 높이기 위해 원점을 지나는 조건을 삭제
 - 쓸모있는 정보를 유지하며 더 작은차원의 벡터들로 선형변환하고자 하는것.
 - 변환후 차원을 맞추기 위해 다시 역변환, 그리고 원래 벡터와 변환후 벡터 비교
